### Installation
This project requires the following Python packages: `numpy`, `pandas`, `matplotlib`, `control`, `sympy`, `ipywidgets` and `bokeh`.
`pip install numpy`
`pip install pandas`
`pip install matplotlib`
`pip install control`
`pip install sympy`
`pip install ipywidgets`
`pip install bokeh`
